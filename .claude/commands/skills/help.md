# Skills Commands

Discover, install, and manage agent skills from the skills.sh ecosystem.

## Instructions

Display the following help information:

```
Skills Management Commands

  /skills:find [QUERY]     Search for skills by keyword or domain
  /skills:add PACKAGE      Install a skill from GitHub or skills.sh
  /skills:list             List installed skills in this project
  /skills:update           Update all installed skills
  /skills:check            Check for available skill updates
  /skills:help             This help message

What are Skills?
  Skills are SKILL.md files that give Claude Code specialized procedural
  knowledge - domain expertise, workflows, and best practices that persist
  across sessions. They are installed per-project via the skills CLI.

Skills CLI:
  npx skills find [query]  Search for skills interactively
  npx skills add <pkg>     Install a skill
  npx skills check         Check for updates
  npx skills update        Update installed skills

Browse skills:  https://skills.sh/
```
